# 智能记事本

## 触发
- 「记一下明天下午开会」「7月11日去吃饭」→ 自动解析日期录入
- 「我车在哪」「车停B2了」→ 停车管理
- 「查看记事本」「打开记事本」→ 月历视图
- 「明天有什么事」「7月10号有什么安排」→ 查询

## MODAL id
`skill-notepad`

## 设计原则
- **极简**：只分停车🅿️和事件•，无勾选、无待办标记
- **全前端渲染**：所有事件数据嵌入 JS，翻月/翻年无需服务端
- **年月选择器**：点标题弹出，任意年月直接跳转

## 脚本
```
# 显示记事本（手机端）
python scripts/notepad.py --show --out mobile
```

## 展示结构
1. 🅿️ 停车卡片（固定在顶部）
2. 📅 月历网格（圆点标记：🟠事件 🔵循环任务）
3. 📌 日详情（点击任意日期切换，含每日循环任务）
4. 导航：‹ › 逐月翻 · 点标题弹出年月选择器

## 数据存储
`beancount/data/notepad.json`

## 视觉风格
与 income-report / fund-realtime 统一：
- 背景 `#0f1923`，卡片 `#1a2a3a`
- 🟠事件圆点 `#ffb347`，🔵循环圆点 `#5a9aff`
- 今天日期蓝色高亮 `#5a9aff`
- `prefers-color-scheme:light` 浅色自适应
